# 小岛AI输入法 · 对标与路线图

对标对象：豆包输入法（AI 语音输入的体验标杆）。本项目差异化：**全本地转写、隐私零上传、开源可改**；AI 能力（润色）用用户自备的大模型 API Key，可插拔。

## 对标豆包输入法：能力差距清单

| 能力 | 豆包输入法 | 小岛AI输入法 | 状态 |

| 中英混说识别 | ✅（云端） | ✅（SenseVoice 本地，zh/yue/en/ja/ko） | 已有 |

## P1（本次已实现）

- **设置系统**：`settings.json`（gitignored）+ 默认值深合并；菜单栏「设置」入口；「重新加载配置」免重启生效。
- **热键可配**：左 Option（默认）/ 右 Option / 右 Command / F19，菜单单选即时生效。
- **AI 润色（可开关，默认关）**：转写后过一遍大模型，去口水词、修同音错字、规范标点；
  - provider `openai`（默认）：任意 OpenAI 兼容端点，DeepSeek / Kimi / GLM / OpenAI / ollama 本地模型均可；
  - provider `anthropic`：Anthropic API（需自备 key）；
  - **fail-open**：润色超时/报错一律回退原始转写，绝不阻断出字。
- **准确率辅助**：`hotwords` 热词表（注入润色提示词纠同音错字）+ `replacements` 离线替换表（不依赖 LLM、零延迟）。

## P2（体验完善）

- 独立 .app 打包（py2app / Platypus）：权限授给 App 本身，双击即用，免终端。
- 首次启动引导：三项权限检测 + 一键跳转对应设置页。
- 润色风格预设：书面 / 口语 / 翻译成英文 / 会议纪要，菜单一键切换（不同 system prompt）。
- 输入历史面板：最近 N 条转写/润色结果，点击复制。
- 录音开始/结束提示音；菜单栏图标动效。

## P3（大功能）

- **流式实时上屏**：transcribe.cpp 支持 streaming，边说边出字（需处理已上屏文本的修正回退，交互复杂度高）。
- 双语模式：说中文出英文（润色 prompt 变体 + 快捷切换）。
- 常驻润色通道：本地小模型（ollama qwen 系）做默认润色，无网络、无 key、~1s。

## 不做

- 云端账号体系、跨设备同步、词库云备份——与"全本地、隐私零上传"的定位冲突。
